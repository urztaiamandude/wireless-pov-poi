# Project Notes

This directory contains ongoing project notes, design decisions, and development context for the Wireless POV Poi system.

## Contents

- [Architecture Decisions](architecture_decisions.md) — Key design choices and their rationale
- [Development Log](development_log.md) — Chronological record of major changes and milestones

## Purpose

These notes serve as a persistent memory for contributors and AI assistants working on the project. They capture context that may not be obvious from code alone — the *why* behind decisions, known trade-offs, and lessons learned during development.

## Contributing

When making significant changes, add a dated entry to `development_log.md` describing:
- What changed and why
- Any trade-offs or alternatives considered
- Known limitations or follow-up work needed
